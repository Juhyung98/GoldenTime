# Golden Time

2020 서울하드웨어 해커톤 프로젝트 출품작  

## 개요
**LPWA를 응용한 **  

위 물음에서 라포미러 프로젝트는 시작되었습니다.  

Rapport : 공감, 친밀감  
Mirror : 거울  

이웃끼리의 공감과 친밀감 향상을 위한 IoT 스마트 미러 솔루션, 바로 라포미러입니다.
